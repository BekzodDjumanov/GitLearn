Workflow Status

A status update in a CI/CD pipeline is crucial, as it provides supplemental information concerning its current state. It states whether a process if failed, successful, or in progress.

Core Indicators

  • Success / Passed: All jobs in the workflow completed without errors.
  • Failure / Failed: At least one job encountered an error or did not complete.
  • In progress / Running: Jobs are currently executing.
  • Canceled / Skipped: Workflow or pipeline was intentionally stopped or certain jobs were skipped.

The Goal

Workflow status provides useful information on software changes.

Summary

Workflow or pipeline status shows the current state of automated processes, indicating whether tasks have passed, failed, are running, or were canceled.
