Log

Logging shows a detailed and comprehensive history of a current branch, letting you track what changes were made, when, and by whom. "git log" is an essential tool in identifying a branches background.

  • View commit messages, authors, timestamps, and hashes.
  • Navigate through commit history with pagination.
  • Filter or format output with options (e.g., --oneline, --graph).

Common Commands

Basic log

$ git log

One-line summary

$ git log --oneline

Visual graph

$ git log --graph --oneline --all

Show history by name

$ git log --author="Name"

Summary

"git log" allows a user to view commit history of a branch.
