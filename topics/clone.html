Clone

"Cloning" allows a user to create a copy or replicate a repository locally.This lets you download the entire project history, branches, and files from a repository hosted on platforms like GitHub, GitLab, or Bitbucket so you can start working on it locally.

  • Creation: creates a new directory named after the repo by default (or a name you specify).
  • Copy: copies branch and commit history.
  • Collaborate: allows users to clone original repositories, allowing cross platform collaboration.

Example

$ git clone https://github.com/username/repository.git

Clone into custom folder

$ git clone https://github.com/username/repository.git my-folder

Summary

"Cloning" is a key component for software collaboration and referencing. It grants developers the ability to copy.
