Init

Git Init (initiation) creates a new repository in your current directory. This sets up the hidden .git folder where Git tracks your files and history.

  • Start Tracking: Turns a regular folder into a Git repository.
  • Local Only: No connection to remote until you add one.
  • One-Time Setup: Usually run once when starting a new project.

Example

$ mkdir my-project
$ cd my-project
$ git init

Summary

Init is a powerful and useful command in any git powered project. It allows the user to initialize a repository that can store platform-independent code.
