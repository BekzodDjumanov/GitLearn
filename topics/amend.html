Amending Commits

the amend feature (git commit --amend) allows users to alter the most recent commit, which can include typos, implement forgotten changes, preserving commit history.

Common Commands

Amending a commit

$ git commit --amend -m "New commit message"

Adding file to last commit

$ git add forgotten-file.txt $ git commit --amend --no-edit

Interactive editing

$ git commit --amend
