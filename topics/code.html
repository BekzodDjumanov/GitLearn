Code Formatting & Visibility

When writing and formatting code, ensure consistent use of indentation, spacing, and style conventions. This allows other developers, whether skilled or beginner to understand and assimilate to the code's purpose.

Helpful Tools

Prettier is a VSCode extension that automatically formats and applies modern styling conventions.

ESLint is a JS plugin that analyzes code and quickly identifies and fixes problems.

ChatGPT is an AI LLM that automatically formats and styles any code employing advanced text recognition.

Visibility relates to how clear and accessible your code and comments are to other developers. GitHub allows users to set access privileges to public or private. Public repositories encourage open collaboration and knowledge sharing, while private repositories protect sensitive or proprietary code. Properly managing repository visibility helps maintain security and fosters effective teamwork by controlling who can view, contribute, or review the codebase.

Summary

Clean and visible code improves collaboration and reduces bugs by making your project easier to understand and maintain.
