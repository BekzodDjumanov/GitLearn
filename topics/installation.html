Git Installation

Before we prepare to delve into the foundational components of Git, follow the guide in order to configure a proper development environment.

Linux

$ sudo apt install git

macOS

Homebrew

$ brew install git

If you dont have Homebrew

$ xcode-select --install

Windows

Please visit

https://git-scm.com/downloads
  • Download the .exe installer.
  • Run the installer and follow the setup wizard (default options are usually fine).

Verify

$ git --version
